---- To install this project on Pc follow the below steps 

----- Step 1: Install the xampp server on your pc 

------Step 2: Once the installation is done open xampp and in the Apache server and mysql

------- Step 3 : Go to Admin section of Mysql in xampp control panel 

--------- Step 4 : Once the phpadmin page is loaded on your browser add the database files from the dataset folder in phpadmin database section

---------- Step 5  : Once the database are created then paste the cargo booking system project in C/xampp/htdocs folder

------------ Step 6 : All Done
